# 🌊 Ocean Flight (WASD + Mouse)

В программе можно летать над красивым океаном. Управление: WASD + мышь.

Этот проект — небольшая доработка шейдера "Seascape" Александра Алексеева. Я добавил управление от первого лица, чтобы можно было свободно исследовать процедурный океан.

🕹️ **[https://github.com/Kenotgg/OceanControll/releases/tag/Release]**

## Управление

- **WASD** — движение вперед/назад/влево/вправо
- **Мышь** — поворот камеры (вид от первого лица)
